Comment ça marche ?

Les hubs

Un hub est un groupe de travail disposant de son propre lien d'invitation via lequel vous pourrez inviter vos élèves.

Depuis le panneau de gestion gérez vos participants, les activités disponibles pour le groupe ou encore les informations du hub.

Les modules

Un module est une activité traitant d'un thème spécifique. Chaque module aborde une thématique différente du domaine informatique et est étudié pour être aussi ludique que pédagogique. Sélectionnez les plus adaptés et organisez en quelques clics les supports accompagnants votre cours.

Proposez un ensemble de ressources interactives à faire en autonomie pour laisser vos élèves découvrir les sujets clés du numérique à leur rythme.
